Naming Conventions

All elements should be named according to the following naming conventions:

Type Naming Convention
Fields/Variables camelCase
Const Fields/Variables ALL_UPPER_CASE_SNAKE_CASE
Properties camelCase if simple getter or setter
PascalCase if performing calculation or operation more advanced than a
simple check or assignment.
See property naming for examples.
Classes PascalCase
Enums PascalCase
Enum elements ALL_UPPER_CASE_SNAKE_CASE
Interfaces PascalCase
Methods PascalCase
Structs PascalCase
Namespaces PascalCase
